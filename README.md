# CustomDialogFragment
DialogFragment弹窗工具

添加依赖

```
//在工程build.gradle(Project:)下添加maven仓库
allprojects {
    repositories {
        google()
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}

//在app的build.gradle(Module:app)下添加依赖
dependencies {
    implementation 'com.github.JinpoHuang:CustomDialogFragment:v1.0'
}
```
